Most commonly, this is accomplished with the display: none; CSS declaration. While this does work, it is semantically incorrect. A better option is to use the hidden Boolean attribute, which is a global attribute available to all elements for use. Functionally it performs the same way as the CSS declaration, but semantically it represents an element that should be hidden, or ignored, for the time being.
